# Netflix-Data-Cleaning-Analysis-and-Visualization
## 📌 Project Overview
This project analyzes Netflix content data to understand trends in movies, TV shows, genres, and content distribution across countries.

## 📊 Dataset
The dataset includes:
- Show ID
- Type (Movie / TV Show)
- Title
- Director
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genres

## 🛠 Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud

## 🔍 Project Workflow
1. Data Cleaning
2. Handling Missing Values
3. Removing Duplicates
4. Exploratory Data Analysis
5. Visualization

## 📈 Visualizations
- Movies vs TV Shows distribution
- Rating frequency
- Top countries producing content
- Content added over time
- Popular genres
- Word cloud of titles

## 🎯 Key Insights
- Movies dominate Netflix compared to TV shows.
- The United States produces the largest amount of Netflix content.
- Content additions increased significantly after 2016.
